## ✅ **Tasks: JavaScript Ternary Operator**

### **1. Even or Odd**

Write a program that checks if a number is even or odd using a ternary operator.

### **2. Adult or Minor**

Given an `age` variable, print `"Adult"` if age ≥ 18, otherwise `"Minor"`.

### **3. Temperature Check**

Given a temperature value:

* Print `"Hot"` if temperature > 30
* Otherwise print `"Cool"`

### **4. Login Status**

Given a boolean `isLoggedIn`, print:

* `"Welcome back!"` if true
* `"Please log in"` if false

### **5. Discount Eligibility**

If a cart total ≥ 100, print `"Discount applied"`, else `"No discount"`.

### **6. Max of Two Numbers**

Given two numbers, use a ternary to print the larger one.

### **7. Valid Username**

Given a `username`, print:

* `"Valid"` if it has at least 4 characters
* `"Too short"` otherwise

### **8. Divisible by 5**

Check whether a number is divisible by 5.
Output `"Divisible"` or `"Not divisible"`.

### **9. Exam Result**

Given a score:

* `"Pass"` if score ≥ 50
* `"Fail"` otherwise

### **10. Light Mode / Dark Mode**

Given a boolean `darkMode`, set a variable `theme` to:

* `"dark"` if true
* `"light"` if false
